# Regional Sales Analysis Dashboard (Tableau)

## 📌 Objective
To compare sales performance across two regions and identify trends to support business decision-making.

## 🛠 Tools Used
- Tableau (Data Cleaning, Visualisation, Interactivity)

## 📊 Dataset
Sales dataset including:
- Regions
- Sales figures
- Time periods

## 🔍 Process
- Cleaned and validated data
- Created comparative visualisations (bar charts, line charts)
- Added filters for region and time analysis
- Designed an interactive dashboard

## 📈 Key Insights
- Identified high-performing regions
- Highlighted seasonal sales trends
- Enabled deeper comparison using filters

## 🚀 Business Impact
- Simplified complex data into visual insights
- Supported strategic regional decisions
- Improved data accessibility for stakeholders
